# Incident Report Builder

A browser-based tool to help structure professional incident response reports.

## Features

- **Who / What / When / Where / Why / How** framework
- T1 (Basic) and T2 (Full) report tiers
- Flexible custom fields - add what you need
- Appendix sections: Timeline, IOCs, MITRE ATT&CK, OSINT, Queries, Evidence
- Validation for required fields
- Copy as Markdown for Notion/docs
- Print-ready output

## Usage

1. Fill in the sections relevant to your investigation
2. Toggle optional/appendix sections as needed
3. Click `[ GENERATE_REPORT ]`
4. Copy as Markdown or Print
